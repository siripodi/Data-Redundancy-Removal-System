# Data Redundancy Removal System

## 📌 Project Description

This project is developed as part of the **CodeAlpha Cloud Computing Internship**.

The Data Redundancy Removal System is a web application built using **Python, Flask, and Firebase Firestore**. It stores user information and prevents duplicate email entries by checking the database before saving new records.

## 🚀 Features

- User Registration Form
- Firebase Firestore Integration
- Duplicate Email Detection
- Prevents Duplicate Data
- Responsive User Interface
- Built with Flask

## 🛠 Technologies Used

- Python
- Flask
- Firebase Firestore
- HTML
- CSS

## ▶️ How to Run

1. Install dependencies

```

```

2. Run the application

```
python app.py
```

3. Open your browser

```
http://127.0.0.1:5000
```
